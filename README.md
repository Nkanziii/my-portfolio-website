# Niki Khanezai — Portfolio Website

A personal portfolio website for Niki Khanezai, creative coder, front-end developer and UI designer currently studying MSc Creative Computing at University of the Arts London.

## Live Site

[nkanziii.github.io](https://nkanziii.github.io/my-portfolio-website/)

## Pages

- **index.html** — Main portfolio page with hero, about, projects, skills and contact sections
- **sketches.html** — p5.js creative coding gallery featuring six interactive generative sketches

## Features

- Animated twinkling stars canvas background
- Custom cursor with trailing ring effect
- Scroll reveal animations
- Responsive layout (desktop, tablet, mobile)
- Scrolling marquee ticker
- Interactive project cards
- p5.js generative art sketches embedded live

## Tech Stack

- HTML5, CSS3, JavaScript (vanilla)
- [p5.js](https://p5js.org/) — generative sketches
- [Three.js](https://threejs.org/) — 3D project
- [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) — Google Fonts
